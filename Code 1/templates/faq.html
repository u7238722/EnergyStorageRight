FAQ

Q1:How to use this website?

Please go to Tutorial and have a check.


Q2:What is this page userd for?

This page is used to display the related renewable energy's data on the world. At the same time, users are allowed to input some parameters and calculated the ROI and IRR on the project if they expect to do a investment.


Q3:Who are the target users?

Our target users are those who are interested in investing renewable energy but not professional enough in this area. We provide basic estimation about the project return rate and hence users can get a general idea easily.


Q4:How can I find the data source?

The data showing in this web site mainly from three sources:
(1)Solar database: https://globalsolaratlas.info/download/world
(2)Wind database: https://globalwindatlas.info/downloads/gis-files
(3)Storage(Hydro) database: https://nationalmap.gov.au/renewables/#share=s-oDPMo1jDBBtwBNhD


Q5:Where do the solar, wind and hydro layer cover?

The layers showing on this web site are from https://nationalmap.gov.au/renewables/
As this is maintained by Australian government and is mainly used for researches on renewable energies in Australia, so the solar and wind layer only covers Australia. The Hydro layer covers the whole world.
